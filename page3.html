input("Enter your username:")

But without storing the value to the variable, we can't use the input value.So let's store the value to user variable

Example Code:

user = input("Enter your user:")
print(f"Hello! {(user)}")

Output:

Enter your user:Emu

#let's say you type Emu

Hello! Emu
[Program Finished]

Now we can reuse the variable again and again, but input() save the value as a string so we can't really use it as integer or float data types.But we can use int(input("")) to store it as integer type and float(input("")) to store it as float type

Example Code:

age = int(input("age:"))
height = float(input("height:"))
print(age)
print(height)

Output:

age:14
height:1.74

#let's type something

1.74
