Variables and Data Types

Variables:

Python use Variable to store information.Variable is like a box where we store items.

Example Code:

city = "Mandalay"

Output:


[Program Finished]

We successfully stored "Mandalay" in variable named city.Variable is useless if we didn't use so let's print out the variable data.

Example Code:

city = "Mandalay"
print(city)

Output:

Mandalay
[Program Finished]

To print out the variable item we can't use qoutes " " and the letters has to be the same.If we use qoutes " " it will become a string

Example Code:

city = "Mandalay"
print("city")

Output:

city
